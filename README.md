# LongCat-Folder
LongCat AI的一个油猴脚本，可以帮助你更好的对聊天记录进行归档
复制上述文件到油猴脚本中，即可使用。

Alt + 鼠标左键 点击聊天记录即可选择加入文件夹，一个聊天记录支持加入多个文件夹中，支持新建文件夹、级联删除、同步、跳转等功能，感兴趣的赶快试试吧。
